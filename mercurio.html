El Mercurio

News Website Redesign

On this project I redesign the website for El Mercurio, the most important newspaper in Cuenca – Ecuador. The aim was to create a modern user-friendly interface to ideally communicate the most important news at the top, and the user can apply a filter to select him/her interest topics. The design of the site has been adapted to the current corporate identity and modify here and there to implement a modern aesthetic to differentiate correctly each section / topic of the site. The whole website is set up using responsive designs across all devices, allowing a consistent usage on smart phone, tablet or desktop computer.
